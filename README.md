# mjpg-streamer-ext

boot.sh
image-diff*.sh
loop-snapshot.sh
mjpg-streamer.sh
rsync-snapshots.sh

## 1. install mjpg-streamer

see [mjpg-streamer git](https://github.com/jacksonliam/mjpg-streamer)

## 2. copy scripts

```bash
$ cd ~
$ mkdir bin
$ git clone https://github.com/ytani01/mjpg-streamer-ext.git
$ cp mjpg-streamer-ext/*.sh ~/bin
```

then login again.
